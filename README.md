# Responsive Product Listing App

This project is a responsive single-page application built using Next.js.

## Live Demo

🚀 **[View Live Demo on Vercel](https://product-store-psi-one.vercel.app/)**

## Getting Started

1. Clone the repository:
   https://github.com/harshajagtap2021-cpu/product-store.git

2. Install dependencies:
   npm install

3. Run the development server:
   npm run dev

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- Fetch products from FakeStore API
- Display product image, title and price
- Responsive grid layout
  - Desktop: 4 columns
  - Tablet: 2 columns
  - Mobile: 1 column
- Newsletter form with Name and Email
- Basic validation
- Submitted data logged to browser console

## API Used
https://fakestoreapi.com/products

Next.js, HTML, CSS
