# Responsive Product Listing App

This project is a responsive single-page application built using Next.js.

## Features
- Fetch products from FakeStore API
- Display product image, title and price
- Responsive grid layout
  - Desktop: 4 columns
  - Tablet: 2 columns
  - Mobile: 1 column
- Newsletter form with Name and Email
- Basic validation
- Submitted data logged to browser console

## API Used
https://fakestoreapi.com/products

## Tech Stack
Next.js, HTML, CSS, JavaScript
